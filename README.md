# RENEWEX - Renewable Energy Investment Marketplace

An educational platform for corporate training in renewable energy investment. This simulation allows users to explore utility-scale renewable energy projects, purchase market intelligence resources, and submit Non-Binding Offers (NBOs).

## Features

- **Project Marketplace**: Browse 14 solar and wind projects across Spain, Portugal, France, Germany, and Italy
- **Resource Services**: Purchase country-specific market intelligence (yield assessments, curtailment studies, PPA origination, etc.)
- **News Feed**: Stay informed about market developments affecting investment decisions
- **Bids Leaderboard**: Track project popularity by number of NBOs submitted
- **User Profile**: Manage purchased documents and submitted NBOs

## Tech Stack

- React 18
- Tailwind CSS
- Lucide React Icons

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/renewex-platform.git
cd renewex-platform
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

## Usage

1. **Landing Page**: Learn about the platform and register/login
2. **Projects Tab**: Browse and filter renewable energy projects
3. **Resources Tab**: Purchase market intelligence documents
4. **News Tab**: Read market updates and regulatory changes
5. **Bids Tab**: View the leaderboard of most popular projects
6. **Profile**: Access your purchased documents and submitted NBOs

## Budget

Users start with a €2,000,000 budget to allocate across resource purchases.
